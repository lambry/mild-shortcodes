#Shorts

A simple set of shortcodes.

If your theme already includes Font Awesome add: `add_theme_support( 'font-awesome' );`

###Main Shortcodes
* Rows/Columns
* Icons
* Buttons
* Panels
* Tabs
* Accordian
* Alignment
* Posts
* Subpages
* Restrict
* Login
* Sitemap
* Google Map
* iFrame

####Supplementary Shortcodes (handy in widgets)
* Image
* Link
